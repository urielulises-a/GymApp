# Manual de Usuario — Gimnasio (Web, solo vistas)

## Acceso
- Abrir la URL en el navegador (no hay login real; pantalla de login es de demostración).

## Módulos
### Dashboard
- KPIs y gráficas con datos de ejemplo.

### Miembros
- Tabla con búsqueda local. Botones Crear/Editar/Eliminar muestran mensajes de demo.

### Planes
- Tabla y formulario de ejemplo (sin guardado real).

### Suscripciones
- Formulario que calcula y **muestra** fecha de fin (no guarda).

### Pagos y Recibo
- Formulario de pago con monto estático; botón “Generar recibo” navega a la vista de **Recibo**.
- Recibo se puede **imprimir/guardar** (PDF) desde el navegador.

### Asistencia
- Búsqueda de miembro y botón “Check‑in” (muestra mensaje de demo).

### Reportes
- Gráficas (ingresos, activos/inactivos, visitas) con datos estáticos.

### Ajustes
- Cambiar tema claro/oscuro (afecta a toda la app).

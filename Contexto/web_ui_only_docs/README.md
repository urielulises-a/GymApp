# Contexto para Cursor — Flutter Web **UI-only**

Arquitectura y guías para generar solo **vistas** (sin backend/persistencia). Orden sugerido:
1. Lee `MARCO_TEORICO.md` (visión).
2. Lee `TECHNICAL_DOCUMENTATION.md` (arquitectura).
3. Copia `pubspec.yaml`, corre `flutter pub get`.
4. Sigue `DEVELOPMENT_GUIDE.md` para run en Chrome.
5. Usa `PROMPTS_CURSOR.md` como guion de tareas para Cursor.
6. Refiere `USER_MANUAL.md` para flujos esperados.
7. Despliega con `DEPLOYMENT_GUIDE.md`.
